# PHP OOP CRUD Application-->

A simple CRUD (Create, Read, Update, Delete) application built using PHP Object-Oriented Programming (OOP) and MySQL.

## Features

* Create Records
* Read Records
* Update Records
* Delete Records
* MySQL Database Integration
* OOP Based Structure
* Reusable Database Classes

## Technologies Used

* PHP
* MySQL

## Database Configuration

Update your database credentials in `database.php`:

```php
private $db_host = "localhost";
private $db_user = "root";
private $db_pass = "";
private $db_name = "test";
```

## Installation

1. Clone the repository

```bash
git clone https://github.com/AD1TYAverma/PHP-oop-project
```

2. Move the project to your server directory

```text
C:/wamp64/www/
```

3. Create a MySQL database

```sql
CREATE DATABASE test;
```

4. Import the SQL file

5. Start Apache and MySQL

6. Open in browser

```text
http://localhost/project-folder
```
